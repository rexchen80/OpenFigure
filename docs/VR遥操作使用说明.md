# VR遥操作使用说明

## 概述

本模块提供VR位姿数据的UDP接收、EMA平滑处理和按键事件检测功能，用于MuJoCo机器人IK控制。支持实时遥操作、动作录制和回放功能。

## 启动前准备

### 1. 网络连接

Quest 和上位机电脑需连接**同一局域网**（同一个 WiFi 或有线网络）。

### 2. 查询上位机 IP 地址

| 系统 | 命令/方式 |
|------|-----------|
| Linux | 终端执行 `ifconfig`，查看对应网卡的 `inet` 地址 |
| Windows | 命令提示符执行 `ipconfig`，查看"IPv4 地址" |
| macOS | 系统设置 → Wi-Fi → 详细信息，查看 IP 地址 |

### 3. 运行上位机程序

```bash
python example/demo_ik_vr.py
```

程序启动后监听 UDP 端口，等待 Quest 发送数据。

## Meta Quest 3 VR 极简操作说明

### IP设置与连接
- 左手摇杆双击：显示/隐藏IP设置界面。
- 使用射线点击IP段的+/−按钮调整数值；点击“连接/Connect”执行连接并保存IP。

### 基础操作
- 右手摇杆双击：开启/关闭追踪控制（开启后头/手跟随，关闭后回到Home位置）。
- Grab/握持键（手柄握持触发键）：按住使能对应手臂跟随手柄，松开停止控制。
- 扳机（Index Trigger）：夹爪开合力度（0–100），按下更闭合，松开张开。

### 复位与校准
- 复位：双击A（右手）复位当前目标到初始姿态。
- 校准：长按右手B键约2秒，完成底座高度/朝向校准并震动反馈。

### 功能总览
- 追踪开关与自动保护：右手摇杆双击切换；两个手柄同时离手超过设定超时后自动关闭并回Home。
- 手臂使能与速度渐增：按住Grab使能手臂跟随，跟随速度在按住期间从低速平滑提升到上限；松开停止。
- 头部跟随与回Home：追踪开启时头部持续跟随；关闭时平滑回到Home姿态。
- 夹爪控制：扳机值映射为0–100的夹爪开合百分比。
- IP设置界面：左摇杆双击切换显示；射线点选+/−修改各段；“连接/Connect”保存并同步到网络模块。
- 目标复位：双击A（右手）将目标物体重置为初始位置/旋转/缩放。
- 底座校准：右手B键长按约2秒，按当前头部/偏移配置校准底座高度与朝向，并震动提示。
- 双击按钮事件与震动：左手X/Y、右手A/B的双击事件被识别并发送，同时触发短震动反馈。
- 位置稳定性与防跳变：开始控制前需满足位置稳定时间与最大位移阈值，避免追踪丢失后的跳变。
- 扭矩调节：右手摇杆上下可调整扭矩目标值，系统平滑逼近目标。
- 追踪状态网络同步：追踪开关变化会同步到网络并重复发送多帧以确保送达。
- 控制器在手检测：仅在手柄被系统追踪（在手中）时保持控制，配合超时保护提升安全性。

## 使用流程（重要）

### 完整操作步骤

```
1. 双击左摇杆     → 打开IP设置界面
2. 射线调节IP段并点Connect → 连接并保存IP
3. 双击右摇杆     → 开启追踪（头/手跟随）
4. 按住握持键     → 使能对应手臂跟随，松开停止
5. 扳机控制夹爪   → 0–100%开合
6. 长按右手B约2秒 → 底座高度/朝向校准并震动反馈
7. 双击A（右）    → 复位当前目标到初始姿态
8. 双击右摇杆     → 关闭追踪并回Home
```

| 操作 | VR手柄 | 键盘 | 功能 |
|------|--------|------|------|
| 设置IP | 双击左摇杆 | - | 显示/隐藏IP设置界面 |
| 调整IP段 | 射线点+/− | - | 修改各段数值 |
| 连接 | Connect按钮 | - | 执行连接并保存IP |
| 追踪开关 | 双击右摇杆 | - | 头/手跟随开启/关闭并回Home |
| 手臂使能 | 按住握持键 | - | 对应手臂跟随，速度渐增，松开停止 |
| 夹爪 | 扳机 | - | 夹爪开合力度0–100 |
