## 📌 项目简介

OpenFigure 是一个开源人形机器人项目，旨在让更多人可以以相对低成本，亲手制造、改装并迭代全尺寸人形机器人。  
希望通过开放设计与社区协作，让大家可以真正「摸得着、改得动、玩得爽」的人形机器人平台，可复现，容易复现。 🤝

## 🧠 核心思路

- **可全 3D 打印 🧩**  
  尽可能通过桌面级 3D 打印机即可完成主要结构件的生产，降低制造门槛。

- **模块化设计 🧱**  
  双臂、腰部、头部、双腿等全部采用模块化解耦设计，可按预算与需求独立装配或升级。

- **开箱即用的系统 ROM 💽**  
  提供适配树莓派 / Jetson 等平台的定制系统 ROM，尽量减少环境搭建的繁琐步骤，一键启动常用功能。

- **VLA 友好兼容 🤖🧠**  
  高度兼容市面常见的 VLA 模型，规划实现一键下载、部署与运行，降低 VLA 体验门槛，让更多人可以尝试。

## 🤝 如何参与

你可以通过多种方式参与 OpenFigure 项目：

- ⭐ **给仓库点一个 Star**，是对项目最大的鼓励

- 🔧 **提交 PR**：修复问题、优化结构、补充文档或 Demo
